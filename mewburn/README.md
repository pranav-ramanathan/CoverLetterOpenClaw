# Mewburn Cover Letter Pack

Files:
- `cover_letter_template.md` — generic markdown template
- `cover_letter_mewburn_ai_ml.md` — tailored markdown draft
- `cover_letter_template.tex` — clean LaTeX version ready to compile

## Compile to PDF

From this folder:

```bash
pdflatex cover_letter_template.tex
```

or (recommended if available):

```bash
latexmk -pdf cover_letter_template.tex
```

Output: `cover_letter_template.pdf`
